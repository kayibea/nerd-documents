
# Documents

This repository contains curated documentation on Linux, shell tools, system administration, security, and more.

## Structure

- **basics:** introductory topics (bash, git, text manipulation)
- **system:** OS and hardware
- **security:** SSH, GPG, hardening
- **storage:** backups and disk management
- **package:** package tools & distro-specific guides
- **scheduling:** cron, systemd-timers
- **advanced:** specialization topics
- **monitoring:** logs & observability
- **security_tools:** tools such as OpenSSL, pass
